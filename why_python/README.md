# Why Python

* Why learn Python over other languages?
  * syntax - simpler and readable
  * extensive libraries and frameworks
  * community support
* How can Python help with Cloud/DevOps/Support Engineering?
  * To integrate with tools/platforms we need to use
  * To automate with repetitive tasks
  * To retrieve and manipulate data from external systems, services, application components
* How does it run?
  * interpreted
  * doesnt need compiling
* What is a venv?
  * Virtual Environment
  * Ensures you have the libraries and the versions of that libraries required for each project
